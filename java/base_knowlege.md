# 指令重排
提升CPU的利用率。

指令1 -> 指令2 -> 指令3

若指令1耗时很长，指令2，指令3耗时很短，那么就可以进行重新排序。


CPU的速度比内存快很多。假设有两个指令，两个指令没有任何关联，第一个指令需要从内存中获取数据，而第二个指令直接就可以执行。

按照正常的顺序来执行，总的执行时间比较长。

由于CPU一般采用流水线来执行指令，流水线是并行的，。。。。。。
